# Reiki Landing Site

## What This Is

Premium landing page для Reiki-практики, направленный на конверсию посетителей в клиентов. Сайт предоставляет полную информацию о практике Reiki, её преимуществах и процессе сеансов, с интегрированной системой онлайн-записи через календарь и контактные формы.

## Core Value

Потенциальные клиенты могут легко узнать о практике Reiki и записаться на сеанс онлайн через удобную интеграцию с календарем бронирования.

### Out of Scope

- Блог или система публикаций — фокус на landing page, не на контент-платформе
- Личный кабинет пользователей — не нужен для простой записи на сеанс
- Онлайн-оплата на сайте — оплата обсуждается отдельно после записи
- CMS для управления контентом — статичный контент определен заранее
- Отзывы клиентов — можно добавить в v2, если будет запрос

## Context

**Целевая аудитория:**

- Люди, интересующиеся практиками релаксации и альтернативными подходами
- Те, кто ищет способы снижения стресса и психо-эмоциональной поддержки
- Пользователи, открытые к духовным и энергетическим практикам

**Ключевые сообщения:**

- Reiki — это практика релаксации и поддержки благополучия через энергетическую работу
- Практика помогает снизить стресс, улучшить эмоциональное состояние, создать внутренний баланс
- Честность о научной оценке: не является медицинским лечением, используется как дополнение
- Безопасность и профессионализм

**Конверсионная воронка:**

1. Посетитель попадает на Hero → впечатление + интерес
2. Читает о практике Reiki → понимание
3. Видит преимущества → желание попробовать
4. Изучает процесс сеанса → снятие опасений
5. Видит цены → принятие решения
6. Записывается через форму/календарь → конверсия

**Технический контекст:**

- Современный веб-стек 2026 года
- Статичная генерация для максимальной производительности
- SEO-first подход для органического трафика
- Мультиязычность для расширения аудитории

## Constraints

- **Tech Stack**: Next.js 15, React 19, Tailwind CSS, TypeScript — современный и производительный стек
- **Hosting**: Vercel или Netlify — бесплатный хостинг с автодеплоем
- **Performance**: Lighthouse score >90 — быстрая загрузка критична для конверсии
- **Accessibility**: WCAG 2.1 AA — доступность для всех пользователей
- **Legal**: Обязательный disclaimer о том, что Reiki не заменяет медицинскую помощь
- **Content Integrity**: Честная информация о научной оценке практики
